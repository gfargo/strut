# strut Skills

Procedural workflows for strut VPS stack management. Each skill provides step-by-step commands for a specific operational domain.

| Skill | Use When |
| --- | --- |
| `vps-deployment` | Deploying to production, releasing, stopping containers |
| `vps-debugging` | Diagnosing production issues, viewing logs, checking status |
| `database-backups` | Backing up data, restoring, syncing environments |
| `key-rotation` | Rotating credentials, security incidents, offboarding |
| `stack-validation` | Creating new stacks, auditing existing ones |
| `drift-detection` | Checking for unauthorized VPS changes, auto-fix |
| `monitoring-setup` | Deploying Prometheus/Grafana, configuring alerts |
| `domain-ssl` | Setting up domains, obtaining/renewing SSL certs |
| `vps-audit-migration` | Onboarding a new VPS, discovering containers |

See `CLAUDE.md` at the project root for the full developer context and CLI reference.
